AgarioServer.py is the server file, which requires the python-twisted installed in the running environment.

use below command to running the server file:
sudo python AgarioServer.py

the port the server listening should not be occupied by other programs.

the server connection is created in the GameScene.m file, connect() method,
befor the connection, the ip address shold be replaced by the current ip address. 

*****NOTICE****************************************
before start the game. please start the sever first
